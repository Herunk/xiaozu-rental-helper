## 项目简介

小租同学是一款基于豆包Agent开发的智能租房助手，专门针对第一次租房的大学生群体设计。它整合了AI图像识别、实时数据搜索和专业租房知识库，提供从找房、验房、价格核验到签约的全流程一站式服务，帮助大学生避开租房陷阱，租到安全、性价比高的房子。

**解决的核心痛点**：
- 第一次租房经验不足，不知道怎么看房、怎么签合同
- 无法识别串串房、甲醛房等高危房源
- 不了解市场行情，容易被房东/中介坑高价
- 看不懂合同条款，容易踩霸王条款陷阱

## 📸 详细功能介绍

### 1. 智能验房系统
**支持格式**：JPG/PNG/WebP照片、PDF合同/房产证扫描件
**分析维度**：基于19个专业评分维度，分为致命问题(-20分)、严重问题(-10分)、一般问题(-5分)
**特色能力**：
- ✅ **串串房专项识别**：通过装修风格、家具细节、新旧程度五维识别串串房，附带甲醛防控建议
- ✅ **PDF合同分析**：自动提取关键条款，标注霸王条款和风险点
- ✅ **结构化评分报告**：输出优势分析、问题分析、建议措施和补充确认事项
- ✅ **PDF报告导出**：一键生成专业的房屋可租性评估报告

**评分输出示例**：
```
房源综合评分：65/100 | 推荐等级：可以考虑

一、优势分析
- 朝南主卧，采光良好
- 距离地铁800米内，通勤方便

二、问题分析
  致命问题（-20分/项）：
  - 疑似串串房：全屋网红风装修，家具做工粗糙
  严重问题（-10分/项）：
  - 3楼无电梯，搬家成本高
  一般问题（-5分/项）：
  - 空调能效3级，电费偏高

三、建议措施
- 要求房东加装防臭地漏后再签约
- 自带甲醛自测盒，关窗密闭12小时后检测

四、补充确认事项
- 房屋位置（精确到小区）、面积、户型
- 是民水民电还是商水商电
- 是房东直租还是二房东
```

### 2. 实时价格核验
**数据来源**：实时搜索贝壳找房、安居客、链家等平台最新数据
**判断逻辑**：
- 租金 < 均价×0.9：价格偏低，提示核实房源真实性
- 租金 > 均价×1.1：价格偏高，建议砍价
- 其他情况：价格合理
**特色能力**：
- ✅ **全国覆盖**：支持所有城市和区域的实时价格查询
- ✅ **凶宅自动查询**：当用户提供具体小区+楼栋号时，自动同步查询凶宅历史
- ✅ **价格对比报告**：生成详细的价格对比结论和砍价建议

### 3. 全流程专业问答
**覆盖7大主题**：
1. 如何租房（渠道、流程、注意事项）
2. 租在哪里（区域选择、通勤、配套）
3. 租房模式（合租vs整租对比）
4. 选哪种房（房屋类型、优缺点）
5. 选房建议（19个雷点、避坑指南）
6. 验房签房（合同条款、押金、费用）
7. 水电燃气（实时价格查询、费用计算）

**专业知识库**：
- 合同条款等级制度（🔴红线/🟡黄线/🟢绿线）
- 19个租房雷点详细说明
- 租房全流程步骤指南
- 押金保护和维权方法

## 🛠️ 技术实现

### 核心技术栈
- **AI平台**： WorkBuddy
- **依赖库**：fpdf2（PDF生成）
- **能力调用**：WebSearch（实时数据）、FileUpload（文件分析）

### 核心脚本说明
1. **`scripts/price_comparison.py`**：租房价格对比计算工具
   - 标准化户型名称
   - 计算价格偏差百分比
   - 生成价格合理性结论和建议

2. **`scripts/generate_report.py`**：PDF可租性报告生成器
   - 自动检测系统中文字体（Windows微软雅黑/macOS苹方/Linux Noto Sans）
   - 自动静默安装fpdf2依赖
   - 生成结构化的PDF报告，包含评分、问题分析和建议
