## 📸 验房分析使用指南

### 支持格式

| 格式 | 用途 | 示例 |
|:--|:--|:--|
| 📷 照片 (.jpg/.png/.webp) | 房源实拍、户型图、房产证截图 | 中介发的照片、自己拍的看房照 |
| 📄 PDF (.pdf) | 租房合同、房屋登记表、房产证扫描件 | 电子合同、扫描版合同 |

可混合上传，AI 自动按类型分组分析。

### 拍哪些照片最有用？

建议按以下顺序上传，覆盖 19 个评分维度：

1. **客厅全景** → 看装修风格（识别串串房）、采光、家电
2. **卧室** → 看朝向、床位数、插座位置
3. **卫生间** → 看是否发霉、地漏防臭、热水器
4. **厨房** → 看油烟机、燃气灶、橱柜清洁度
5. **楼道/门口** → 看卫生、门禁安全

### 评分怎么看？

```
房源综合评分：85/100 | 推荐等级：强烈推荐

≥80 分 → 强烈推荐：综合条件优秀
60-79 分 → 可以考虑：有小问题但可接受
<60 分 → 不推荐：问题较多，建议继续找
```

### 能识别哪些问题？

**致命问题（-20 分/项，出现即不推荐）：**
- 串串房（网红风装修 + 密度板家具 + 新装）
- 非电网直缴电表（被房东加价）
- 隔音极差 / 电器噪音大
- 周边有工地

**严重问题（-10 分/项）：**
- 一楼 / 阴面房间 / 商水商电
- 电器能效 3 级以上
- 落地窗低楼层隐私差
- 卫生间反味

**一般问题（-5 分/项）：**
- 核心电器缺失 / 电器不干净
- 水电由房东代缴 / 楼道卫生差
- 楼下有餐饮店 / 插座不足 / 阳台未封闭

### PDF 报告

验房分析完成后，系统会询问你是否需要 PDF 房源可租性评估报告。报告包含：
- 品牌页眉 + 生成时间
- 大字体评分 + 颜色分级标签
- 问题分析（致命红 / 严重橙 / 一般黄）
- 建议措施 + 补充确认事项

说「生成 PDF」即可获取。

## 💰 价格核验使用指南

### 怎么用？

直接告诉我：**城市 + 区域 + 户型 + 租金**

```
你：重庆渝北区一室一厅 1800/月，贵不贵？
```

系统会实时搜索同区域最新租金，给出结论。

### 判断标准

| 价格 vs 市场均价 | 结论 | 行动建议 |
|:--|:--|:--|
| < 90% | 🔴 偏低 | 警惕虚假房源、二房东诈骗 |
| 90%-110% | 🟢 合理 | 结合综合条件做决定 |
| > 110% | 🟡 偏高 | 尝试砍价或考虑其他房源 |

### 凶宅查询

如果你提供了**具体小区名 + 楼栋号**，系统会自动同步查询凶宅历史。结果会融入价格报告一并告知，不需要额外操作。

## 📋 合同审查使用指南

### 怎么用？

上传合同 PDF，系统自动提取关键条款，按三级分类审查：

| 等级 | 含义 | 行动 |
|:--|:--|:--|
| 🔴 红线条款 | 绝对不能签的霸王条款 | 要求删除或修改，否则不签 |
| 🟡 黄线条款 | 需要协商关注的条款 | 根据自身需求谈判 |
| 🟢 绿线条款 | 对租客有利的加分项 | 尽量争取写入合同 |

### 常见红线条款示例

- 提前解约扣全部押金 + 剩余租金
- 无条件扣除 30% 以上押金
- 租期内房东可单方面涨租
- 不允许换锁
- 押金超过 2 个月租金

## 📋 核心规则速查

| 规则 | 说明 |
|:--|:--|
| 🔴 安全底线 | 绝对不推荐二房东的房子 |
| 🏠 房屋类型 | 优先推荐普通住宅（民水民电） |
| 📍 选房区域 | 优先公司 3-5 公里内、近地铁的小区 |
| 💰 价格实时 | 每次价格核验必须实时搜索，严禁使用缓存 |
| 🔌 水电实时 | 每次水电查询必须实时搜索用户城市价格 |
| 💬 语言风格 | 使用大学生易懂的口语，避免专业术语 |
| ⚠️ 信息准确 | 不确定的信息明确标注，不编造 |
