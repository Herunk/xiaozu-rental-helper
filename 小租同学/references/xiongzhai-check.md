# 凶宅历史查询方法论

> 本文档说明凶宅历史查询的执行标准、搜索策略和结果判断方法。所有查询必须**实时执行**，严禁使用内置数据或主观判断。

---

## 一、什么是"凶宅"

在中国房地产市场，"凶宅"通常指发生过**非正常死亡事件**的房屋，包括但不限于：

| 事件类型 | 说明 |
|---|---|
| 自杀 | 在房屋内自杀身亡 |
| 谋杀 | 在房屋内被杀害 |
| 意外死亡 | 煤气中毒、火灾、高坠等意外事故致死 |
| 恶性事件 | 入室抢劫致死、恐怖袭击等 |

**以下情况不属于凶宅范畴**（但仍建议告知用户）：
- 自然死亡（老死、病死）
- 正常医疗事故死亡
- 社区公共区域（非房屋内）的事件

---

## 二、查询触发条件

满足以下**所有条件**时，主动触发凶宅查询：

1. 用户提供的地址能定位到**具体小区**
2. 用户提供了**楼栋号或单元号**（仅小区名不够精确）
3. 用户正处于选房/确认房源阶段

**示例触发场景**：
- "我看了XX花园3栋2单元这套房"
- "XX小区1号楼怎么样"
- "这个地址：YY路ZZ小区B栋""

**不触发场景**：
- 用户仅提供区域名（如"成都高新区"）
- 用户仅提供小区名但无楼栋号
- 用户处于"找房"阶段，尚未锁定具体房源

## 三、搜索策略

### 搜索渠道（按优先级）

| 优先级 | 渠道 | 搜索方式 | 说明 |
|:---:|---|---|---|
| 1 | 微信公众号 | `site:mp.weixin.qq.com` | 大量本地化凶宅信息 |
| 2 | 微博 | `site:weibo.com` | 网友曝光和讨论 |
| 3 | 新闻网站 | `site:news.163.com OR site:thepaper.cn` | 重大事件报道 |
| 4 | 地方论坛 | 百度贴吧、地方社区 | 本地居民讨论 |
| 5 | 搜索引擎 | 通用搜索 + 百度 | 综合信息覆盖 |

### 搜索关键词模板

**基础搜索**（必执行）：
```
"<小区全名> 凶宅"
"<小区全名> <楼栋号> 死亡"
"<城市> <小区名> 发生过 重大事件"
```

**扩展搜索**（发现线索时执行）：
```
"<小区名> <门牌号> 自杀"
"<小区名> 命案"
"<小区名> 煤气中毒 死亡"
"<小区名> 火灾 死亡"
```

### 精确搜索格式

当用户提供完整地址时，使用精确搜索：
```
"<省> <市> <区> <小区名> <楼栋> <单元> 凶宅"
"<小区名> <楼栋> 单元 死亡 自杀"
```

## 四、结果判断标准

### 判断等级

| 等级 | 结果 | 条件 | 输出颜色/标记 |
|---|:---:|---|:---:|
| A | ✅ 暂无记录 | 多渠道搜索均**未发现**相关记录 | 绿色 |
| B | ⚠️ 需关注 | 搜索发现**存在相关记录** | 黄色/橙色 |
| C | ⚠️ 信息不足 | 小区信息不精确，无法确定 | 灰色 |

### 等级A处理（暂无记录）

```
查询结果：✅ 暂无记录

本次查询在以下渠道均未发现该房屋的重大事件记录：
- 微信公众号（本地资讯）
- 微博/社区论坛
- 新闻网站
- 地方贴吧

⚠️ 注意：网络公开信息可能存在遗漏，建议：
1. 实地向物业或邻居了解情况
2. 签约前在合同中增加"凶宅告知条款"
3. 综合判断后再做决定
```

### 等级B处理（需关注）

```
查询结果：⚠️ 需关注

搜索发现以下相关记录（信息来源附后）：

【记录摘要】
<时间> 在 <具体位置> 发生了 <事件类型>
来源：<平台> <链接>

【核实建议】
1. 要求房东/中介提供书面说明
2. 可向物业查询房屋档案
3. 建议综合考虑后谨慎决定
4. 如有顾虑可直接放弃该房源
```

### 等级C处理（信息不足）

```
查询结果：⚠️ 信息不足

目前提供的信息不足以完成凶宅查询：
缺少：<具体缺少的信息，如楼栋号>

请提供更精确的地址信息：
- 小区全称（避免使用简称或别名）
- 具体楼栋号 + 单元号（或门牌号）

信息越精确，查询结果越可靠。
```

## 五、免责声明模板

每次凶宅查询报告必须附带以下免责声明：

```
━━━━━━━━━━━━━━━
⚠️ 免责声明

本报告基于网络公开信息查询，结果仅供参考。
网络信息可能存在以下局限：
- 部分事件未被网络记录
- 历史久远的事件可能难以查到
- 信息可能存在误差或不完整

建议结合以下方式综合判断：
1. 实地向物业/邻居了解
2. 在合同中增加凶宅告知条款
3. 通过当地派出所核实（部分城市支持）

如有任何疑虑，建议直接放弃该房源，另选其他房源。
━━━━━━━━━━━━━━━
```

## 六、注意事项

### ❌ 禁止行为

1. **禁止主观判断**：不得因房屋价格便宜、装修差等主观因素暗示"可能是凶宅"
2. **禁止夸大信息**：不得将未经证实的信息作为结论告知用户
3. **禁止强制建议**：不替用户做决定，只提供信息和建议
4. **禁止使用内置数据**：本功能无任何内置凶宅数据库，必须实时搜索

### ✅ 正确行为

1. 每次查询都必须**实时执行** WebSearch
2. 所有信息来源必须**注明平台和链接**
3. 查询结果必须使用**统一格式输出**
4. 无论结果如何都必须附带**免责声明**
5. 发现相关记录时，**优先建议用户核实**，而非直接建议放弃

## 七、补充：凶宅相关法律知识

### 凶宅的法律地位

- **《民法典》相关规定**：出卖人/出租人有义务如实告知房屋的重大瑕疵，包括凶宅信息
- **隐瞒凶宅信息的法律后果**：若房东故意隐瞒，可主张撤销合同并要求赔偿
- **合同条款建议**：建议用户在租房合同中加入"凶宅告知条款"，要求房东书面确认

### 凶宅条款示例（供用户参考）

```
甲乙双方确认，甲方已如实告知乙方该房屋的以下情况：
□ 该房屋未发生过非正常死亡事件
□ 该房屋未发生过重大安全事件

若甲方未能如实告知，乙方有权提前解除合同并要求甲方退还全部租金及押金，
同时赔偿乙方一个月租金作为违约金。
```
