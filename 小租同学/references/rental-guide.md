# 租房全流程指南

## 一、如何租房（租房方式和流程）

### 1.1 租房渠道对比

| 渠道 | 优点 | 缺点 | 推荐指数 |
|------|------|------|----------|
| 房东直租 | 无中介费，价格低，沟通直接 | 房源少，需要自己找 | ★★★★★ |
| 品牌中介（链家、我爱我家） | 房源多，有保障，合同规范 | 中介费高（通常1个月房租） | ★★★★ |
| 小中介 | 中介费可谈 | 房源可能不真实，存在虚假房源 | ★★ |
| 长租公寓（自如、蛋壳） | 省心，装修好，有管家 | 租金溢价高，服务费 | ★★★ |
| 豆瓣租房小组 | 房源真实，个人房源多 | 需要自己筛选，警惕假房源 | ★★★★ |
| 闲鱼/58同城/安居客 | 房源多，覆盖城市广 | 虚假房源和中介伪装房源多，须仔细甄别 | ★ |

### 1.2 常用租房 App 功能要点

**贝壳找房（推荐）**
- 房源相对真实：经纪人实名认证，房源有实勘照片
- **价格核验功能**：App 内置同小区历史成交价，可与房东报价对比参考
- **VR 看房**：支持线上 VR 实景看房，节省时间
- **筛房功能**：支持按地铁线/商圈/价格/户型精细筛选
- 注意：平台经纪人带看会产生中介费，签合同前确认费用明细

**安居客**
- 房源量大，覆盖城市广，三四线城市房源较多
- **低价引流套路**：首页常出现标价极低的房源，吸引电话联系，实际看房时告知"已租出"并推荐其他房源，警惕此类虚假低价
- **筛选功能**：支持按装修/楼层/朝向/配套筛选，但同一房源可能由多个中介发布，信息重复较多
- **中介识别**：标注"个人"标签的房源真实性相对较高；中介房源通常有"经纪人"标识
- 注意：平台中介质量参差不齐，建议优先联系评分高、成交量大的经纪人

### 1.2 租房流程

1. **确定预算**：月租不超过月收入的 1/3
2. **确定区域**：优先考虑通勤距离（3-5 公里内最佳）
3. **筛选房源**：线上初步筛选，列出意向房源
4. **实地看房**：至少看 3-5 套再做决定
5. **核实信息**：查看房本、房东身份证、确认产权
6. **谈判签约**：争取优惠，仔细阅读合同条款
7. **交接验收**：拍照留证，记录房屋设施状况
8. **入住交接**：确认水电燃气表底数，移交钥匙

### 1.3 注意事项

- **看房建议白天+晚上各一次**：白天看采光、装修质量；晚上听噪音（邻居/街道/楼下商铺）
- 入住前要求房东结清之前的水电燃气费
- 所有口头承诺必须写入合同
- 不要急于交定金，至少看完 3 套再做决定
- **房源性价比过高须警惕**：租金明显低于市场价→大概率虚假房源或二房东套路

## 二、租在哪里（租赁区域选择）

### 2.1 区域选择原则

1. **通勤优先**：公司到住处的通勤时间控制在 45 分钟以内
2. **地铁沿线**：优先选择地铁站 500 米以内的小区
3. **配套完善**：周边有超市、菜市场、医院、银行等生活设施
4. **安全性**：避免过于偏僻或治安不好的区域
5. **性价比**：在预算范围内选择面积和条件最合适的

### 2.2 区域选择技巧

- 用地图工具搜索"公司地址 + 5 公里"，圈定候选区域
- 对比不同区域同类型房源的均价，判断性价比
- 关注区域的未来发展（地铁规划、商业配套等）
- 了解区域的人口密度和居住氛围
- 询问已在附近租房的同学或同事的真实感受

## 三、租房模式（合租 OR 整租）

### 3.1 合租 vs 整租对比

| 对比项 | 合租 | 整租 |
|--------|------|------|
| 租金 | 低（分摊费用） | 高（独担全部费用） |
| 隐私 | 一般（共享公共区域） | 好（完全独立） |
| 社交 | 有室友可以互相照应 | 独立生活 |
| 矛盾 | 可能产生室友矛盾 | 无室友矛盾 |
| 适合人群 | 预算有限、刚毕业的大学生 | 预算充足、重视隐私 |
| 注意事项 | 合租人背景、生活习惯匹配 | 安全、费用独立 |

### 3.2 合租注意事项

- 了解合租人的基本信息（职业、作息习惯、是否为常住人员）
- **室友背景核验**：警惕经常流动的人员合住（安全隐患和财物风险）
- 确认公共区域的使用规则（厨房、卫生间、客厅）
- 确认费用分摊方式（水电燃气、网费、物业费）
- **室友随意带陌生人留宿**：需提前约定边界，否则安全隐患极大
- **共用设施维修责任**：合同中明确公共卫生间的热水器/洗衣机、厨房煤气灶/冰箱的维修由谁负责

### 3.3 整租注意事项

- 确认房屋面积和户型是否符合需求
- 确认租金包含的费用和不包含的费用
- 注意独居安全问题（门锁、监控、周边治安）

## 六、验房签房（合同+押金+费用）

> **详细内容请参考** `references/contract-clauses.md`
>
> 包含：3条必加合同条款、押金保护措施、水电燃气费用标准

**验房清单**：
1. 检查所有电器是否能正常运行（空调、冰箱、洗衣机、热水器）
2. 检查墙面是否发霉、脱落
3. 测试水龙头水压和下水是否通畅
4. 检查门窗是否能正常开关
5. 记录水电燃气表底数，拍照留证

**签合同必做**：
1. 要求查看房东房产证和身份证
2. 确认所有口头承诺写入合同
3. 明确押金金额和退还条件
4. 约定提前解约条款（建议30天通知）
5. 约定维修责任条款（自然损耗房东负责）

## 七、附录：常见问题速查

| 问题 | 答案 |
|------|------|
| 押金一般多少？ | 1个月租金，超过2个月要警惕 |
| 预付多久房租？ | 建议押一付一，最多押一付三 |
| 中介费多少？ | 通常1个月租金，品牌中介可谈 |
| 怎么看电表？ | 国家电网电表为民用电，房东自装电表可能是加价电 |
| 民水民电多少钱？ | 电约0.5元/度，水约5元/吨（各城市不同，需实时查询） |
