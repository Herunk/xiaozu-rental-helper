### 4. 费用全景与预算计算

当用户表达费用/预算相关意图时，执行以下**单次计算流程**（不依赖网络，运算即时完成）：

**执行流程**（严格按序）：

1. **意图识别**：从用户输入中提取关键参数
   - 月租金（必填，用户未提供则主动询问）
   - 城市（选填，用于判断是否含取暖费）
   - 租房渠道（选填，用于确定中介费系数）
   - 房屋面积（选填，用于估算物业费）
   - 付款方式（选填，默认为押一付三）

2. **城市自动分类**（无需用户说明）：
   - 北方城市（北京/天津/石家庄/太原/呼和浩特/沈阳/大连/长春/哈尔滨/济南/青岛/郑州/西安/兰州/西宁/银川/乌鲁木齐/唐山/秦皇岛/邯郸/保定/大同/包头/鞍山/抚顺/吉林/齐齐哈尔/大庆/洛阳/咸阳等）→ 加算取暖费
   - 南方城市 → 无取暖费

3. **渠道系数映射表**（实时查表，不搜索）：
   | 用户说 | 中介费系数 |
   |:--|:--|
   | 房东直租/个人房东/豆瓣/闲鱼 | 0（免中介费） |
   | 链家/我爱我家/贝壳/中介 | 0.5-1.0（默认0.5，标注可谈） |
   | 自如/泊寓/品牌公寓/长租公寓 | 0.5-1.0（一次性服务费） |
   | 未说明 | 0.5（默认值，提示用户确认） |

4. **一键计算公式**：
   ```
   首月总费用 = 押金 + 预付租金 + 中介费 + 物业费月均 + 取暖费月均 + 网费 + 预存杂费
   
   其中：
   押金 = 月租金 × 1
   预付租金 = 月租金 × N（N=1 押一付一，N=3 押一付三，默认3）
   中介费 = 月租金 × 渠道系数
   物业费月均 = 面积 × 3元/㎡（默认估算，提示用户核实）
   取暖费月均 = 面积 × 25元 ÷ 4（仅北方城市）
   网费月均 = 100元（默认估算）
   预存杂费 = 350元（默认估算，含水电气预存300 + 门禁卡50）
   ```

5. **输出格式**（快速预算版，一次输出全部）：

```
🧮 首月租房总预算

| 费用项 | 金额 | 说明 |
|:--|--:|:--|
| 月租金 | X,XXX | 你的目标月租 |
| 押金 | X,XXX | 押一付三（1个月） |
| 预付租金 | X,XXX | 一次性付3个月 |
| 小计（租金+押金） | X,XXX | 最大头 |
| 中介费 | X,XXX | 渠道系数×月租（可谈空间X-XXX） |
| 物业费（首月） | XXX | 估算值，以合同为准 |
| 取暖费（月均） | XXX | 仅北方城市，合同须明确谁承担 |
| 网费（首月） | 100 | 房东已装则直接付，未装需+初装费100-200 |
| 水电燃气预存 | 300 | 房东可能要求，入住时拍照留底 |
| 门禁卡/钥匙 | 50 | 一次性，退租可退 |
| **首月总计** | **X,XXX** | 准备好这个数就行 💰 |
```

6. **联动提示**（预算输出后，自动衔接其他功能）：
   - 如用户提供了城市+区域 → 追问：「需要我帮你查这个区域的租金行情，看看这个预算能租到什么样的吗？」
   - 如用户未说明渠道 → 追问：「你是通过中介找的、品牌公寓、还是直接联系房东？这会影响中介费。」
   - 费用项中标 ⚠️ 的 → 提示：「物业费、取暖费须在合同中明确由谁承担，口头承诺不算。需要我帮你生成一份费用条款核对清单吗？」

7. **费用条款核对清单生成**（用户需要时生成）：
   基于 `references/expense-checklist.md` 和 `references/contract-clauses.md`，生成签约时可逐条对照的费用条款清单，格式：

```
📋 租房合同费用条款核对清单

签约前逐项确认（打钩通过）：
□ 押金：___元（不超1个月，合同写明"无损坏无欠费全额退还"）
□ 租金：___元/月，付款方式：押___付___
□ 中介费：___元（已确认渠道和金额）
□ 物业费：___元/月，由___承担（合同明确）
□ 取暖费：___元/供暖季，由___承担（北方必写）
□ 网费：___元/月，初装费___元（已确认有无欠费）
□ 水电燃气：民水民电 / 商水商电，表底数已拍照
□ 清洁费：退租自行清洁，不另收保洁费（合同明确）
□ 违约金：提前退租___元，提前收房___元（合同明确）
□ 门禁卡/钥匙：___张，押金___元/张
□ 其他费用：无 / 有___（逐项列出）
```

**预算输出性能优化规则**：
- 预算计算**纯本地运算**，不发起任何网络请求，毫秒级出结果
- 费用清单数据**预加载**自 `references/expense-checklist.md`，不重复搜索
- 渠道系数**内存查表**，无需询问用户即可默认推定
- 仅当用户追问水电燃气具体单价时，才启动实时搜索（遵循核心规则#7）
- 费用清单与合同条款自动关联：用户在费用核对清单中标记的项，同步对应到合同审查的 🔴🟡🟢 体系


### 租房预算表

当用户需要预算表时，基于「费用全景」模块的单次计算流程，生成完整的首月/月度总支出明细表。预算计算**纯本地运算**，无需网络请求，秒出结果。

包含：押金、预付租金、中介费（按渠道区分）、物业费、取暖费（自动识别北方城市）、网费、水电燃气预存、门禁卡、隐藏费用提示。

输出格式参考 `references/expense-checklist.md` 第六节的快速参考表结构，同时标注每一项费用是否需要在合同中明确。
